# conversor-svg
Codigo para interpretar archivos SVG simples. Interpreta SVG simples, limpios, y genera una estética especifica para utilizar en animaciones de las presentaciones que armo. Tiene una croma definida que utilizo mucho. Aplica también un pequeño efecto de resalte y zoom si hay formas cerradas en el svg. Se tiene que copiar el codigo del SVG y pegar en la columna izquierda del modulo, a la derecha aparecerá el SVG con la estética definida por el estilo del código.
